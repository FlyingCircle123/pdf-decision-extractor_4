# 🦆 WHITE CROW — PDF Tool with ATTITUDE

Extract decisions, actions, and insights from any PDF.  
**With 8 weird modes because boring is overrated.**

---

## ✨ What It Does

Upload any PDF (scanned or digital).  
WHITE CROW extracts text and uses AI to find:

| Category | What It Finds |
|----------|---------------|
| 🎯 **Decisions** | Key choices, conclusions, judgments |
| ⚡ **Action Items** | Tasks, next steps, things to DO |
| 💡 **Key Points** | Important ideas, concepts, insights |

**Then it rewrites them in your chosen vibe.**

---

## 🎭  Weird Modes

| Mode | What It Does |
|------|--------------|
| **Normal** | Professional extraction (boring but useful) |
| **ELI5** | Explains like you're 5 (simple metaphors) |
| **Haiku** | Every insight becomes 5-7-5 poetry |
| **Sarcastic** | "Oh wow, how surprising..." energy |
| **Conspiracy** | Nothing is what it seems. WAKE UP. |
| **Motivational** | LET'S GO! CRUSH IT! ABSOLUTE LEGEND! |
